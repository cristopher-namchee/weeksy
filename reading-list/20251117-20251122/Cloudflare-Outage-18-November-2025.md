- Cloudflare outage on 18 November 2025 caused massive outage to internet infrastructure.
- The main cause of this outage is misconfiguration on Cloudflare Bot Management System which is overloaded by feature files due to new system.
- The feature files are then propagated to all network replicas
- A global kill switch might help mitigate this issue faster
